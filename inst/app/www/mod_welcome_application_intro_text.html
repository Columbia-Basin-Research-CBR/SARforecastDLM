SARforecastDLM can be used to explore wild, spring/summer Snake River Chinook salmon, Oncorhynchus tshawytscha, one-year ahead forecasted survival based on changes in indices of ocean upwelling.


This application uses Dynamic Linear Modelling (DLM) to forecast salmon survival using time varying parameters such as, smolt- to- adult survival ratio (SAR) and coastal upwelling indices, Coastal Upwelling Indice (CUI) and Coastal Upwelling Transport Indice (CUTI).


To explore the application, select the Forecast survival page in the side panel, or continue to the Overview section to learn more about how this application can assist is addressing knowledge-action gaps.

Please note that this Shiny App is dependent on data availability and serves as an exploratory tool.
